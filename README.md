# forest-solidity

An implementation of forest in solidity.

```shell
npm install @MASDXI/forest-solidity
```

### Forest

`Forest` is UTXO and DAG inspired.

### ForestV2

`ForestV2` using same mechanism as `Forest` but `ForestV2` fully support ERC-1155 interface.
